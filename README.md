# منصة العقارات الرائدة

منصة عقارات شاملة تجمع بين المشترين والبائعين والمستأجرين بطريقة آمنة وموثوقة.

## 🚀 الميزات الرئيسية

### للمستخدمين
- 🏠 عرض آلاف العقارات المتنوعة
- 🔍 بحث متقدم وتصفية حسب المواصفات
- 📱 تواصل مباشر مع المالكين عبر الواتس والهاتف
- ❤️ حفظ العقارات المفضلة
- 📊 تحليلات توصيات ذكية

### لمالكي العقارات
- ➕ إضافة عقارات بسهولة
- 📈 إحصائيات تفصيلية للعقارات
- 💰 خطط اشتراك مرنة
- 📧 رسائل من المهتمين
- 🎯 عرض محسّن للإعلانات

### للإدارة
- 👥 إدارة المستخدمين والمالكين
- 🏢 مراجعة واعتماد العقارات
- 📊 تحليلات شاملة للمنصة
- 💳 إدارة الدفع والاشتراكات
- 📈 تقارير أداء مفصلة

## 🛠️ البنية التقنية

### Frontend
- **Framework**: Next.js 16
- **UI Components**: Shadcn/UI
- **Styling**: Tailwind CSS
- **State Management**: React Hooks
- **Charts**: Recharts

### Backend
- **Runtime**: Node.js
- **API**: Next.js API Routes
- **Database**: PostgreSQL
- **Authentication**: Session-based with HTTP-only cookies
- **Password Hashing**: PBKDF2

### Database Schema
- **Users**: حسابات المستخدمين والإدارة
- **Properties**: بيانات العقارات
- **Units**: الوحدات والأجزاء
- **Payments**: سجل الدفعات
- **Subscriptions**: الاشتراكات والخطط
- **Sessions**: جلسات المستخدمين
- **Analytics**: تحليلات الاستخدام
- **Favorites**: العقارات المفضلة
- **Market Trends**: اتجاهات السوق

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- PostgreSQL 12+
- npm أو pnpm

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository>
cd real-estate-platform
```

2. **تثبيت المكتبات**
```bash
pnpm install
```

3. **إعداد قاعدة البيانات**
```bash
# تشغيل migration
psql -U postgres -d real_estate -f scripts/01-init-db.sql
```

4. **متغيرات البيئة**
```bash
# .env.local
DATABASE_URL=postgresql://user:password@localhost:5432/real_estate
NODE_ENV=development
```

5. **تشغيل المشروع**
```bash
pnpm dev
```

الموقع سيكون متاحاً على: `http://localhost:3000`
